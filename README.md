# UC-IMU-pointer

To use:
1. Copy code.py and lib to the ESP32 Feather
2. Change the following line to the port your Feather is connected to:
  <PORT = "/dev/tty.usbmodem487F30FE67E91">
3. Run display_cursor.py via local machine.
4. On Feather boot up, hold the device still for 5 seconds to calibrate before moving.

Tips:
Horizontal position is controlled by sensor roll.
